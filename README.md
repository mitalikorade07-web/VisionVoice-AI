# 👁️ VisionVoice AI
🚀 AI-powered voice-first navigation assistant for visually impaired users using real-time computer vision and Murf AI
## 🧠 Overview
VisionVoice AI is a real-time assistive application that uses **computer vision + voice interaction** to help visually impaired users navigate safely.



The system detects objects using AI and provides **live voice guidance**, making interaction completely **hands-free and voice-driven**.



---



## 🎯 Problem Statement



Visually impaired individuals face difficulty navigating safely due to lack of real-time environmental awareness.



---



## 💡 Solution



VisionVoice AI solves this problem by:



- Detecting surroundings using AI  

- Providing real-time voice feedback  

- Enabling full interaction using voice commands  



---



## 🔥 Key Features



### 👁️ Object Detection

- Detects real-world objects using AI (COCO-SSD)



### 🧭 Navigation Guidance

- Gives directions:

  - Move left

  - Move right

  - Stop



### 📏 Distance Awareness

- Estimates object distance:

  - Near

  - Far

  - Very Close



### 🎤 Voice Commands

- Fully voice-controlled system

- Examples:

  - "Start navigation"

  - "What do you see?"

  - "Where should I go?"



### 🧠 AI Chat (Context-Aware)

- Answers user queries based on real-time camera input



### 🔊 Murf AI Integration

- Converts responses into natural human-like voice



### 🌍 Multilingual Support

- Supports:

  - 🇺🇸 English

  - 🇮🇳 Hindi



### 🔄 Flip Camera

- Switch between front and rear camera



### 🔁 Continuous Navigation Mode

- Real-time guidance without user interaction



### 🚨 Emergency Feature

- Triggers alert system



---
