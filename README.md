👁️ VisionVoice AI

🚀 A real-time AI voice assistant that sees, understands, and guides users using vision + speech.

🎯 VisionVoice AI is a voice-first assistive application designed for visually impaired users. It uses real-time computer vision and Murf AI Text-to-Speech to provide instant audio guidance about surroundings.

---

🧠 Overview

VisionVoice AI combines computer vision + voice interaction to create a completely hands-free navigation experience.

The system:

- Detects objects using AI (TensorFlow + COCO-SSD)
- Understands user voice commands
- Responds with real-time voice using Murf AI
- Provides navigation guidance like "move left/right"

👉 This makes interaction fully voice-driven and accessible

---

🚨 Problem Statement

Visually impaired individuals face difficulty understanding their surroundings in real time.

VisionVoice AI solves this by acting as a real-time AI assistant that:

- Detects objects
- Gives spoken guidance
- Enables independent movement

---

💡 Solution

VisionVoice AI solves this problem by:

- Detecting surroundings using AI
- Providing real-time voice feedback
- Enabling full interaction using voice commands

---

🔥 Key Features

👁️ Object Detection

- Detects real-world objects using AI (COCO-SSD)

🧭 Navigation Guidance

- Gives directions:
  - Move left
  - Move right
  - Stop

📏 Distance Awareness

- Estimates object distance:
  - Near
  - Far
  - Very Close

🎤 Voice Commands

- Fully voice-controlled system
- Examples:
  - "Start navigation"
  - "What do you see?"
  - "Where should I go?"

🧠 AI Chat (Context-Aware)

- Answers user queries based on real-time camera input

🔊 Murf AI Integration

- Converts responses into natural human-like voice

🌍 Multilingual Support

- Supports:
  - 🇺🇸 English
  - 🇮🇳 Hindi

🔄 Flip Camera

- Switch between front and rear camera

🔁 Continuous Navigation Mode

- Real-time guidance without user interaction

🚨 Emergency Feature

- Triggers alert system
